## 📌 Featured Work

### 🔹 KORVI · WhatsApp-Native AI Sales Agent
AI agent platform that answers, qualifies, and closes customers for African SMEs directly inside WhatsApp, with Paystack payments in the chat.
- Full CTO-level architecture: Node.js, TypeScript, PostgreSQL with pgvector, Redis, BullMQ, GPT-4o
- 105-ticket engineering backlog, designed and executed solo
- WhatsApp Business API integration via BSP, Kubernetes deployment

👉 **Status:** In active build toward MVP

### 🔹 Causal Inference at National Scale · One Acre Fund RCT
Analysis of a 7-arm randomized controlled trial covering **462,885 farmers** in Nigeria, isolating true intervention effects from correlation.

👉 **Why it matters:** Gold-standard experimental methodology applied at a scale few analysts ever touch

### 🔹 IoT Water Monitoring System · Lagos State
Real-time telemetry pipeline (MQTT, ThingsBoard, Flask) with live dashboards and anomaly alerting for water infrastructure.

👉 **Impact:** Production visibility into distribution for a utility serving one of Africa's largest cities

14-node n8n workflow that parses, scores, and ranks every incoming CV with GPT-4o, turning manual screening into same-day shortlists.

👉 **Impact:** Removes an entire screening bottleneck from recruitment operations
